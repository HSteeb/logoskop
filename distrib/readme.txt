-----------------------------
1. Folders in Logoskop/distrib

_exe
  Used for creating installer

_screenshots
  for use in download web pages

_Versions
  contains installed files for each version (e.g. 3.0.2)
  not uninst.exe
  not Logoskop_Installer.exe
  not multiple installers with varying .twd languages

_Years
  contains installers for each year (e.g. 2010)
  all Logoskop versions, but only latest of same installer name
  all language / International variations
  not installed files
  
-----------------------------
2. Procedure for releasing new installer

- unpack to _Versions/3.0.0 (drop uninst.exe + .twd files)
- move installer to _Years/2010
