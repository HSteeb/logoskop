Creating a new language

Creating a new language (e.g. Afrikaans)

in w32res

  • menu Build - Configurations, add "Release Afrikaans" and "Debug Afrikaans"

in menu Project-Settings, Common tab (for Debug and Release target):

  • temporary files: ../Obj/Release/w32res/
  • output files: ../Release

in menu Project-Settings, Linker tab:

  • output file name: ../Release/w32resafr.dll

in menu Project-Settings, Ressources tab:

  • Name of ressource file: ../Obj/Release/w32res/w32resafr.res
  • Language: (select from list; if not contained, enter in small window "Ressource options" at the bottom of the dialog: "/l 0xNNN" where NNN is a combined language/sublanguage number from VC98\Include\WinNT.h - at best, check against values used in other targets...)
  • Defines: AFX_RESSOURCE_DLL,AFX_TARG_AFR (language code), AFX_TARG_NEU, AFX_TARG_NEUD (for language neutral ressource parts)

in Ressource view:

  • multi-select ressources needed (look which ones are used for other languages, too), menu Edit-Copy
  • menu Insert - "Ressource copy" (for languages not supported in the combobox now appearing: edit w32los.rc directly: copy the text for a given language to an new language, enter LANG_xx SUBLANG_yy from VC98\Include\WinNT.h)

in menu View - Ressource includes:

  • copy mystic statements into area "compile-time directives" (with correct language/codepage) (must probably be moved to the respective language section in w32los.rc????)

Store IDS_XLAT_* strings in UTF-8!
